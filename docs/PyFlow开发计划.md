## 第4周：屏幕拼接功能

**阶段目标**：实现多屏幕拼接模式，通过鼠标穿越屏幕边界切换控制不同电脑

**相关文档**：屏幕拼接功能设计文档.md

### 核心概念

| 概念   | 说明                 |
| ---- | ------------------ |
| 拼接模式 | 鼠标穿越边界切换电脑，而非镜像同步  |
| 对齐方式 | 左对齐、右对齐、上对齐、下对齐    |
| 拼接方向 | 被控端位于主控端的左/右/上/下方向 |
| 坐标映射 | 不同分辨率间按比例映射坐标      |

│   ├── event_dispatcher.py           # 事件分发器（第2周新增）

├── network/
│   ├── __init__.py
│   ├── tcp_server.py                # TCP服务端
│   └── tcp_client.py                # TCP客户端
├── server/                           # 主控端
│   ├── __init__.py
│   ├── mouse_listener.py            # 鼠标事件监听器（第2周新增）
│   ├── share_server.py              # 共享服务端（第2周新增）
│   ├── keyboard_listener.py         # 键盘事件监听器（第3周新增）
│   ├── clipboard_monitor.py          # 剪贴板监控器（第3周新增）
│   ├── screen_edge.py               # 屏幕边缘检测（第3周新增）
│   └── screen_layout/               # 屏幕拼接模块（第4周新增）
│       ├── __init__.py
│       ├── screen_layout.py         # 屏幕布局数据类
│       ├── layout_manager.py        # 布局管理器
│       ├── edge_detector.py        # 边界检测器
│       ├── coordinate_mapper.py    # 坐标映射器
│       └── switch_controller.py     # 切换控制器（第4周新增）
├── client/                           # 被控端
│   ├── __init__.py
│   ├── mouse_controller.py          # 鼠标控制器（第2周新增）
│   ├── share_client.py              # 共享客户端（第2周新增）
│   ├── keyboard_controller.py       # 键盘控制器（第3周新增）
│   ├── clipboard_sync.py            # 剪贴板同步（第3周新增）
│   └── reconnect.py                  # 自动重连（第3周新增）
├── tests/
│   ├── __init__.py
│   └── test_logger.py               # 日志单元测试
└── docs/
    ├── PyFlow开发计划.md

    ├── 第一周开发总结.md
    ├── 第二周开发总结.md
    └── 第三周开发总结.md
