## 项目文件结构

```
pyflow/
├── main.py                           # 程序入口
├── config.py                         # 配置管理
├── requirements.txt                  # 依赖清单
├── build.spec                        # PyInstaller 打包配置
├── config_layout.json                # 屏幕布局配置示例
├── config.ini.example               # 通用配置示例
│
├── common/                           # 通用模块
│   ├── __init__.py
│   ├── logger.py                     # 日志工具
│   ├── system_utils.py               # 系统工具（光标隐藏/显示）
│   ├── protocol.py                   # 核心协议（8种消息类型）
│   ├── event_dispatcher.py           # 事件分发器（支持粘包+变长消息）
│   └── mouse_predictor.py           # 鼠标预测器
│
├── network/                          # 网络层
│   ├── __init__.py
│   ├── tcp_server.py                # TCP服务端（TCP_NODELAY）
│   └── tcp_client.py                # TCP客户端（TCP_NODELAY + 自动重连）
│
├── server/                           # 主控端模块
│   ├── __init__.py
│   ├── share_server.py              # 主控端核心（切换逻辑 + 钩子集成 + 剪贴板双向同步）
│   ├── mouse_hook.py                # Windows WH_MOUSE_LL 低级鼠标钩子（第5周新增）
│   ├── mouse_listener.py            # pynput 鼠标监听
│   ├── keyboard_listener.py         # pynput 键盘监听（支持 suppress 远程抑制）
│   ├── clipboard_monitor.py         # 剪贴板监控（带防回环机制）
│   ├── screen_edge.py               # 屏幕边缘检测
│   └── screen_layout/               # 屏幕布局与切换
│       ├── __init__.py
│       ├── screen_layout.py         # 屏幕数据类
│       ├── layout_manager.py        # 布局管理器
│       ├── edge_detector.py        # 边界检测器
│       ├── coordinate_mapper.py    # 坐标映射（EDGE_OFFSET=15）
│       └── switch_controller.py    # 切换状态机
│
├── client/                           # 被控端模块
│   ├── __init__.py
│   ├── share_client.py              # 被控端核心（事件分发 + 剪贴板监控 + 防回环）
│   ├── mouse_controller.py         # pynput 鼠标模拟
│   ├── keyboard_controller.py      # pynput 键盘模拟
│   ├── clipboard_sync.py           # 剪贴板写入同步
│   └── reconnect.py                # 自动重连
│
├── tests/                           # 测试
│   ├── __init__.py
│   └── test_logger.py               # 日志单元测试
│
└── docs/                            # 文档
    ├── PyFlow开发计划.md            # 本文档
    ├── PyFlow用户手册.md            # 用户手册
    ├── PyFlow（Python版Deskflow）开发计划及技术架构文档.md
    ├── 屏幕拼接功能设计文档.md
    ├── 鼠标切换优化分析与修改方案.md
    ├── 鼠标切换逻辑重构方案.md
    ├── BugReport_切换后鼠标不动_v19.md
    ├── 后续优化计划.md              # 后续优化计划
    ├── 第一周开发总结.md
    ├── 第二周开发总结.md
    ├── 第三周开发总结.md
    ├── 第四周开发总结.md
    ├── 第五周开发总结.md            # 鼠标切换重构
    ├── 第六周开发总结.md            # 键盘抑制 + 剪贴板双向同步
    └── 打包指南.md
```
