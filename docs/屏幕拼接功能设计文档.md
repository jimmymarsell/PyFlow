# PyFlow 屏幕拼接功能设计文档

## 1. 背景与目标

### 1.1 当前实现的问题

目前的实现是"镜像模式"：主控端和被控端的鼠标同步移动，但两台电脑的分辨率不同（主控 2560x1600，被控 1920x1080），导致鼠标移动不跟手、鼠标位置不对应等问题。

### 1.2 目标

实现"屏幕拼接模式"：将多台电脑的屏幕在逻辑上拼接成一个大的虚拟屏幕，用户可以通过移动鼠标穿越屏幕边界来在不同电脑之间切换，实现无缝的多电脑控制体验。

---

## 2. 核心概念

### 2.1 术语定义

| 术语 | 说明 |
|-----|------|
| **主控端** | 用户直接操作的那台电脑，连接着用户的鼠标和键盘 |
| **被控端** | 被主控端控制的远程电脑 |
| **拼接组** | 由主控端和多个被控端组成的一组电脑，它们在逻辑上拼接在一起 |
| **虚拟屏幕** | 由拼接组中所有屏幕拼接而成的逻辑大屏幕 |
| **屏幕网格** | 拼接组中各屏幕的排列布局 |
| **切换边界** | 相邻屏幕之间的边界线，鼠标穿过时触发切换 |

### 2.2 屏幕网格坐标系

```
虚拟屏幕坐标系 (x, y)
┌─────────────────────────────────────────┐
│                                         │
│    ┌───────────────┐   ┌───────────┐    │
│    │   被控端 2    │   │  被控端 3  │    │
│    │   (1920×1080) │   │ (2560×1440)│   │
│    └───────────────┘   └───────────┘    │
│                                         │
│    ┌───────────────────────────────┐     │
│    │         主控端                │     │
│    │       (2560×1600)             │     │
│    │                               │     │
│    └───────────────────────────────┘     │
│                                         │
└─────────────────────────────────────────┘

实际坐标示例：
- 主控端左上角: (100, 200)，右上角: (2660, 200)
- 被控端2左上角: (-1920, 200)，右下角: (0, 1280)
- 被控端3左上角: (2660, -240)，左下角: (2660, 1200)
```

## 3. 拼接对齐方式 (Alignment Mode)

### 3.1 对齐方式类型

| 对齐方式 | 说明 | 示意图 |
|---------|------|--------|
| **左对齐 (Left)** | 被控端屏幕拼接在主控端左侧 | `被控端 │ 主控端` |
| **右对齐 (Right)** | 被控端屏幕拼接在主控端右侧 | `主控端 │ 被控端` |
| **上对齐 (Top)** | 被控端屏幕拼接在主控端上方 | `┌─────────┐`<br>`│  被控端  │`<br>`├─────────┤`<br>`│  主控端  │`<br>`└─────────┘` |
| **下对齐 (Bottom)** | 被控端屏幕拼接在主控端下方 | `┌─────────┐`<br>`│  主控端  │`<br>`├─────────┤`<br>`│  被控端  │`<br>`└─────────┘` |

### 3.2 水平拼接 (左/右对齐)

```
左对齐示例：
                    主控端
           ┌───────────────────┐
           │                   │
           │      (0,0)        │
           │                   │
           └───────────────────┘
                    │
被控端              │  ← 切换边界 (x=0)
┌─────────────┐     │
│             │     │
│  (-w,0)     │     ▼
│             │     │
└─────────────┘

切换规则：
- 鼠标从主控端左边界 (x=0) 向左移动 → 鼠标消失，出现在被控端右边界对应位置
- 鼠标从被控端右边界 (x=-1) 向右移动 → 鼠标消失，出现在主控端左边界对应位置
```

**坐标映射公式（左对齐）：**
```
主控端 → 被控端：
  被控端_x = -被控端宽度 + 主控端_x * (被控端宽度 / 主控端宽度)
  被控端_y = 主控端_y * (被控端高度 / 主控端高度)

被控端 → 主控端：
  主控端_x = 被控端_x + 主控端宽度 * (主控端宽度 / 被控端宽度)  ← 比例映射
  主控端_y = 被控端_y * (主控端高度 / 被控端高度)
```

### 3.3 垂直拼接 (上/下对齐)

```
上对齐示例：
┌─────────────┐
│   被控端    │
│  (0,-h)     │
└─────────────┘
      ▲ 切换边界 (y=0)
      │
┌─────────────┐
│   主控端    │
│  (0,0)      │
└─────────────┘

切换规则：
- 鼠标从主控端上边界 (y=0) 向上移动 → 鼠标消失，出现在被控端下边界对应位置
- 鼠标从被控端下边界 (y=-1) 向下移动 → 鼠标消失，出现在主控端上边界对应位置
```

**坐标映射公式（上对齐）：**
```
主控端 → 被控端：
  被控端_x = 主控端_x * (被控端宽度 / 主控端宽度)
  被控端_y = -被控端高度 + 主控端_y * (被控端高度 / 主控端高度)

被控端 → 主控端：
  主控端_x = 被控端_x * (主控端宽度 / 被控端宽度)
  主控端_y = 被控端_y + 主控端高度 * (主控端高度 / 被控端高度)
```

## 4. 拼接方向与布局

### 4.1 布局配置参数

每台被控端需要配置以下参数：

```python
@dataclass
class ScreenLayout:
    """屏幕布局配置"""
    screen_id: str              # 屏幕唯一标识
    host: str                  # 被控端 IP
    port: int                  # 被控端端口

    # 尺寸信息
    width: int                 # 屏幕宽度 (像素)
    height: int                 # 屏幕高度 (像素)

    # 拼接配置
    direction: str             # 拼接方向: 'left', 'right', 'top', 'bottom'
    alignment: str             # 对齐方式: 'left', 'right', 'top', 'bottom'
    offset_x: int = 0          # X方向微调偏移 (用于精确对齐)
    offset_y: int = 0          # Y方向微调偏移

    # 切换边界配置
    edge_margin: int = 0       # 边缘触发阈值 (像素)
    edge_behavior: str = 'jump' # 切换行为: 'jump' 立即切换, 'warp' 渐变过渡
```

## 5. 鼠标切换流程

### 5.1 切换状态机

```
                    ┌─────────────────┐
                    │   NORMAL        │ ← 主控端正常控制
                    │  (本地鼠标)       │
                    └────────┬────────┘
                             │
              鼠标移动到边界   │
             (x < edge_margin)
                             ▼
                    ┌─────────────────┐
                    │   SWITCHING     │ ← 准备切换
                    │  (隐藏本地鼠标)  │
                    └────────┬────────┘
                             │
                    发送坐标到被控端  │
                    被控端接管鼠标   │
                             ▼
                    ┌─────────────────┐
                    │   REMOTE        │ ← 被控端控制中
                    │  (远程鼠标)      │
                    └────────┬────────┘
                             │
              鼠标移动到边界   │
             (x > width - edge_margin)
                             ▼
                    ┌─────────────────┐
                    │   RETURNING     │ ← 准备返回
                    │  (隐藏远程鼠标)  │
                    └────────┬────────┘
                             │
                    发送坐标到主控端  │
                    主控端接管鼠标   │
                             ▼
                    ┌─────────────────┐
                    │   NORMAL        │ ← 返回正常状态
                    │  (本地鼠标)       │
                    └─────────────────┘
```

### 5.2 坐标映射算法

```python
def map_coordinates(
    from_screen: ScreenLayout,
    to_screen: ScreenLayout,
    x: int, y: int
) -> tuple[int, int]:
    """
    将坐标从源屏幕映射到目标屏幕
    """
    if to_screen.direction == 'left':
        # 源屏幕右边界 → 目标屏幕左边界
        ratio_x = x / from_screen.width
        ratio_y = y / from_screen.height
        new_x = 0  # 目标屏幕左边界
        new_y = int(y * to_screen.height / from_screen.height)
        # 根据对齐方式调整
        if to_screen.alignment == 'left':
            new_x = 0
        elif to_screen.alignment == 'right':
            new_x = to_screen.width - int(x * to_screen.width / from_screen.width)

    elif to_screen.direction == 'right':
        # 源屏幕左边界 → 目标屏幕右边界
        ratio_x = x / from_screen.width
        new_x = to_screen.width - 1  # 目标屏幕右边界
        new_y = int(y * to_screen.height / from_screen.height)

    elif to_screen.direction == 'top':
        # 源屏幕下边界 → 目标屏幕上边界
        ratio_y = y / from_screen.height
        new_x = int(x * to_screen.width / from_screen.width)
        new_y = to_screen.height - 1  # 目标屏幕下边界

    elif to_screen.direction == 'bottom':
        # 源屏幕上边界 → 目标屏幕下边界
        ratio_y = y / from_screen.height
        new_x = int(x * to_screen.width / from_screen.width)
        new_y = 0  # 目标屏幕上边界

    return new_x + to_screen.offset_x, new_y + to_screen.offset_y
```

### 5.3 边界检测

```python
class EdgeDetector:
    """屏幕边界检测器"""

    def __init__(self, screen: ScreenLayout):
        self.screen = screen
        self.edge_margin = screen.edge_margin
        self.direction = screen.direction

    def is_at_edge(self, x: int, y: int) -> bool:
        """检测鼠标是否在切换边界上"""
        margin = self.edge_margin

        if self.direction == 'left':
            return x <= margin
        elif self.direction == 'right':
            return x >= self.screen.width - margin
        elif self.direction == 'top':
            return y <= margin
        elif self.direction == 'bottom':
            return y >= self.screen.height - margin

        return False

    def should_switch(self, x: int, y: int) -> bool:
        """判断是否应该触发切换"""
        if not self.is_at_edge(x, y):
            return False

        # 进一步检查鼠标是否正在向边界方向移动
        # (可以通过速度向量判断)
        return True
```

### 7.2 核心类设计

```python
class LayoutManager:
    """布局管理器 - 管理所有屏幕的布局配置"""

    def __init__(self, config_path: str):
        self.screens: dict[str, ScreenLayout] = {}
        self.local_screen: ScreenLayout = None
        self._load_config(config_path)

    def add_screen(self, screen: ScreenLayout):
        """添加屏幕到布局"""

    def get_screen_at(self, x: int, y: int) -> ScreenLayout:
        """根据坐标查找对应屏幕"""

    def get_adjacent_screen(self, from_screen: ScreenLayout, direction: str) -> ScreenLayout:
        """获取指定方向相邻的屏幕"""

    def calculate_virtual_position(self, screen: ScreenLayout, x: int, y: int) -> tuple[int, int]:
        """计算在虚拟屏幕中的绝对坐标"""


class SwitchController:
    """切换控制器 - 管理屏幕间切换逻辑"""

    def __init__(self, layout_manager: LayoutManager, mouse_controller):
        self.layout_manager = layout_manager
        self.mouse_controller = mouse_controller
        self.current_screen: ScreenLayout = None
        self.state = 'NORMAL'

    def on_mouse_move(self, x: int, y: int):
        """处理鼠标移动事件"""

    def switch_to_screen(self, target_screen: ScreenLayout, x: int, y: int):
        """切换到目标屏幕"""

    def return_to_local(self, x: int, y: int):
        """返回本地屏幕"""


class CoordinateMapper:
    """坐标映射器"""

    @staticmethod
    def map_to_remote(
        from_screen: ScreenLayout,
        to_screen: ScreenLayout,
        x: int, y: int
    ) -> tuple[int, int]:
        """将坐标映射到远程屏幕"""

    @staticmethod
    def map_to_local(
        from_screen: ScreenLayout,
        to_screen: ScreenLayout,
        x: int, y: int
    ) -> tuple[int, int]:
        """将坐标映射回本地屏幕"""
```
