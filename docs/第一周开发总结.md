# PyFlow 第一周开发总结

## 项目概述

**PyFlow** 是基于 Python 开发的跨终端键盘鼠标控制工具（Python 版 Deskflow），采用 C/S 架构，实现局域网内键鼠共享。

### 项目定位

- 学习性质项目，代码模块化、易读
- 对标 Deskflow 官方功能
- 轻量化版本（非工业级）

## 关键技术点说明

### 1. 虚拟环境

**作用**：隔离项目依赖，避免不同项目版本冲突

**常用命令**：

```bash
# 创建虚拟环境
conda create -n pyflow_env python=3.14

# 激活虚拟环境
conda activate pyflow_env

# 安装依赖
pip install -r requirements.txt
```

### 2. requirements.txt

**精确版本 vs 非精确版本**：

```txt
# 精确版本（推荐团队协作）
pynput==1.8.1

# 非精确版本（更灵活）
pynput>=1.7.6
```

**导出当前环境所有包**：

```bash
pip freeze > requirements.txt
```

### 3. bytes 类型

socket 传输只能使用 bytes，不能用 str：

```python
# str 转 bytes
msg_bytes = "Hello".encode("utf-8")

# bytes 转 str
msg_str = data.decode("utf-8")

# 发送
socket.sendall(b"Hello Server!")  # b 前缀表示 bytes

# 接收
data = socket.recv(1024)  # 返回 bytes
```

### 4. TCP socket 关键参数

| 参数/方法                 | 作用          |
| --------------------- | ----------- |
| `AF_INET`             | IPv4 地址族    |
| `SOCK_STREAM`         | TCP 协议      |
| `SO_REUSEADDR`        | 地址复用，避免端口占用 |
| `bind((host, port))`  | 绑定地址和端口     |
| `listen(backlog)`     | 开始监听        |
| `accept()`            | 接受客户端连接（阻塞） |
| `recv(buffer_size)`   | 接收数据（阻塞）    |
| `sendall(data)`       | 发送所有数据      |
| `settimeout(seconds)` | 设置超时        |

## 遇到的问题及解决方案

### 问题1：Windows 文件占用

**现象**：

```
PermissionError: [WinError 32] 另一个程序正在使用此文件
```

**原因**：Windows 文件锁机制比 Linux/macOS 更严格

**解决方案**：

```python
def tearDown(self):
    logger = logging.getLogger()
    for handler in logger.handlers[:]:
        handler.close()
        logger.removeHandler(handler)

    if os.path.exists(LOG_DIR):
        for file in os.listdir(LOG_DIR):
            if "test" in file:
                try:
                    os.remove(os.path.join(LOG_DIR, file))
                except PermissionError:
                    pass  # Windows 文件锁保险处理
```

### 问题2：日志文件命名

**现象**：测试文件名与 logger.py 生成的不一致

**原因**：logger.py 中 timestamp 格式为 `%Y%m%d_%H`（包含小时），而测试硬编码了 `%Y%m%d`

**解决方案**：

```python
# 测试中动态获取
from datetime import datetime
timestamp = datetime.now().strftime("%Y%m%d_%H")
log_file = os.path.join(LOG_DIR, f"{timestamp}_test.log")
```

## 第2周预备知识

### 协议设计基础

PyFlow 使用自定义二进制协议，关键点：

- 使用 `struct` 模块打包/解析二进制数据
- 鼠标移动事件：8字节（1字节类型 + 4字节X + 4字节Y）
- 键盘事件：6字节（1字节类型 + 2字节按键码 + 1字节状态 + 2字节保留）
- 剪贴板事件：可变长度（1字节类型 + 2字节长度 + N字节内容）

### struct 模块基础

```python
import struct

# 打包（大端序）
data = struct.pack(">ii", x, y)  # > = 大端序, i = int32

# 解包
x, y = struct.unpack(">ii", data)

# 格式符
">" = 大端序
"<" = 小端序
"i" = int32
"h" = int16
"B" = uint8
```

### pynput 基础

```python
from pynput import mouse, keyboard

# 监听鼠标
def on_move(x, y):
    print(f"鼠标移动: {x}, {y}")

listener = mouse.Listener(on_move=on_move)
listener.start()

# 模拟鼠标移动
mouse.Controller().position = (x, y)
mouse.Controller().move(dx, dy)

# 监听键盘
def on_press(key):
    print(f"按键: {key}")

listener = keyboard.Listener(on_press=on_press)
listener.start()

# 模拟键盘
keyboard.Controller().press(key)
keyboard.Controller().release(key)
```
