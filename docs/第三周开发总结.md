# PyFlow 第3周开发总结

## 项目概述

**PyFlow** 是基于 Python 开发的跨终端键盘鼠标控制工具（Python 版 Deskflow），采用 C/S 架构，实现局域网内键鼠共享。

## 功能说明

### 1. 键盘事件监听 (server/keyboard_listener.py)

| 类/方法                                        | 说明         | 参数                        | 返回值 |
| ---------------------------------------------- | ------------ | --------------------------- | ------ |
| `KeyboardListener`                             | 键盘监听器类 | -                           | -      |
| `KeyboardListener.__init__(on_keyboard_event)` | 初始化监听器 | on_keyboard_event: 回调函数 | -      |
| `KeyboardListener.start()`                     | 启动监听     | 无                          | -      |
| `KeyboardListener.stop()`                      | 停止监听     | 无                          | -      |

使用 pynput 的 keyboard.Listener 监听键盘按下和释放事件，通过 pack_keyboard 打包后发送。

### 2. 键盘事件模拟 (client/keyboard_controller.py)

| 类/方法                                          | 说明           | 参数                    | 返回值 |
| ------------------------------------------------ | -------------- | ----------------------- | ------ |
| `KeyboardController`                             | 键盘控制器类   | -                       | -      |
| `KeyboardController.press(key_code)`             | 模拟按键按下   | key_code: 按键码        | -      |
| `KeyboardController.release(key_code)`           | 模拟按键释放   | key_code: 按键码        | -      |
| `KeyboardController.type_key(key_code, pressed)` | 执行按键或释放 | key_code, pressed: bool | -      |

### 3. 剪贴板监控 (server/clipboard_monitor.py)

| 类/方法                                                         | 说明               | 参数                                               | 返回值 |
| --------------------------------------------------------------- | ------------------ | -------------------------------------------------- | ------ |
| `ClipboardMonitor`                                              | 剪贴板监控器类     | -                                                  | -      |
| `ClipboardMonitor.__init__(on_clipboard_change, poll_interval)` | 初始化监控器       | on_clipboard_change: 回调, poll_interval: 轮询间隔 | -      |
| `ClipboardMonitor.start()`                                      | 启动监控           | 无                                                 | -      |
| `ClipboardMonitor.stop()`                                       | 停止监控           | 无                                                 | -      |
| `ClipboardMonitor.get_current_content()`                        | 获取当前剪贴板内容 | 无                                                 | str    |

使用 pyperclip.paste() 轮询检测剪贴板变化，支持可配置的轮询间隔。

| 类/方法                                   | 说明               | 参数              | 返回值 |
| ----------------------------------------- | ------------------ | ----------------- | ------ |
| `ClipboardSync`                           | 剪贴板同步器类     | -                 | -      |

| `ClipboardSync.get_current_content()`     | 获取当前剪贴板内容 | 无                | str    |

### 5. 屏幕边缘检测 (server/screen_edge.py)

| 类/方法                                       | 说明             | 参数                                       | 返回值 |
| --------------------------------------------- | ---------------- | ------------------------------------------ | ------ |
| `ScreenEdge`                                  | 屏幕边缘检测器类 | -                                          | -      |
| `ScreenEdge.__init__(on_edge_hit, threshold)` | 初始化检测器     | on_edge_hit: 回调, threshold: 边缘阈值像素 | -      |

## 技术要点

1. **键盘事件处理**：使用 key.vk 获取虚拟键码，兼容普通键和特殊键
2. **剪贴板轮询**：使用 pyperclip.paste() 定期检查剪贴板变化
3. **边缘检测**：基于 pyautogui.position() 获取鼠标位置，判断是否接近屏幕边缘
4. **重连策略**：指数退避算法，从初始延迟 1s 最大增加到 30s，支持无限重试
