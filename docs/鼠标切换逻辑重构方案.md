# PyFlow 鼠标无缝切换重构方案

## 1. 目标体验

**一把键盘一个鼠标控制多台电脑，像物理多屏一样丝滑：**

- 鼠标在主控电脑屏幕上正常操作，被控电脑**完全不受影响**
- 鼠标移到主控端屏幕边界（如右边缘），光标"滑过去"出现在被控端屏幕上（如左边缘）
- 此时主控端光标隐藏+键盘抑制，所有鼠标/键盘操作作用在被控端
- 在被控端屏幕对向边界（如右边缘）移出，光标"滑回来"出现在主控端对应位置
- **同一时刻只有一台电脑在接收输入**

## 2. 核心逻辑

没有"模式"概念，就是**本地控制**和**远程控制**两个状态：

```
本地控制 → 鼠标碰到边缘 → 远程控制
远程控制 → 虚拟光标碰到对向边缘 → 本地控制
```

**本地控制状态：**
- 主控端正常使用鼠标/键盘
- **不发送任何鼠标/键盘事件到被控端**
- 被控端收到任何事件都忽略

**远程控制状态：**
- 主控端光标隐藏 + 键盘抑制（suppress=True）
- Windows：WH_MOUSE_LL 钩子拦截鼠标，计算增量并发送；非 Windows：pynput 回中重定位
- 鼠标移动以增量（delta）方式发送给被控端
- 键盘事件发送给被控端执行
- 点击/滚动事件发送给被控端（不含绝对坐标，在光标当前位置执行）
- 服务端追踪虚拟光标位置（_remote_x, _remote_y），检测对向边缘自动返回
- 被控端收到事件后执行

## 3. 数据流

### 3.1 本地控制

```
鼠标移动 → MouseListener
             ├─ 边缘检测 (ScreenEdge.update)
             └─ 不发送到被控端

键盘事件 → KeyboardListener (suppress=False)
             └─ 不发送到被控端

剪贴板 → ClipboardMonitor
           └─ 检测变化 → 发送到被控端 (防回环)
```

### 3.2 触发切换

```
边缘命中 → ScreenEdge.on_edge_hit
         → SwitchController.switch_to_remote
             ├─ CoordinateMapper.map_to_remote(边界坐标, EDGE_OFFSET=15)
             ├─ hide_cursor()
             ├─ keyboard_listener.set_suppress(True)
             ├─ SetCursorPos(中心点) + 重置 delta 基准
             ├─ MouseHook.set_active(True) [Windows] 或 pynput 回中
             ├─ 发送 SWITCH(mapped_x, mapped_y, 0, remote_w, remote_h)
             └─ 状态 = REMOTE

             同时：
             ├─ _switch_grace_until = now + 500ms (防弹跳)
             └─ _remote_x = mapped_x, _remote_y = mapped_y (追踪虚拟光标)
```

### 3.3 远程控制

```
鼠标移动 (Windows) → MouseHook callback → 计算 dx/dy
                     ├─ 发送 MOUSE_DELTA(dx, dy)
                     ├─ 更新 _remote_x/y (按比例缩放)
                     └─ _check_remote_edge()

鼠标移动 (非Windows) → MouseListener._on_mouse_event
                     ├─ 计算 dx = x - prev_x, dy = y - prev_y
                     ├─ 发送 MOUSE_DELTA(dx, dy)
                     ├─ 更新 _remote_x/y
                     ├─ 检测到对向边缘 → 返回本地
                     └─ 靠近屏幕边缘 → _reposition_cursor()

点击事件 → MouseHook/MouseListener
           └─ 发送 MOUSE_BUTTON(button, pressed, 0, 0)

滚动事件 → MouseHook/MouseListener
           └─ 发送 MOUSE_SCROLL(0, 0, dx, dy)

键盘事件 → KeyboardListener (suppress=True)
           └─ 转发到被控端

剪贴板 → ClipboardMonitor (检测变化, 防_last_remote_clipboard回环)
          └─ 发送到被控端
```

### 3.4 返回本地

```
虚拟光标到达对向边缘 → _check_remote_edge() → should_return=True
                      → CoordinateMapper.map_to_local(remote_x, remote_y)
                      → SwitchController.return_to_local(local_x, local_y)
                          ├─ keyboard_listener.set_suppress(False)
                          ├─ show_cursor()
                          ├─ move_cursor_to(local_x, local_y)
                          ├─ MouseHook.set_active(False) [Windows] 或 重启 MouseListener
                          └─ 状态 = NORMAL
```
